# Contributing

## Quick start

```sh
pnpm setup
```

## Run apps

```sh
pnpm dev:admin
pnpm dev:mobile
pnpm dev:api
```

## Common tasks

```sh
pnpm lint
pnpm type-check
pnpm build
```

## Notes

- Use pnpm for installs and scripts.
- Run commands from the repo root (the folder with `apps/` and `packages/`).
